# SpacePresser

Костыль, чтобы быстро жать `пробел` при просмотре видео с помощью [ViewerYUV](https://github.com/The220th/yuv_simple_java_viewer).

# Использование

## Перед запуском

С помощью `pip3` нужно установить пакет `pyautogui`:

``` bash
> pip3 install pyautogui
```

## Пример запуска:

``` bash
> python3 SpacePresser.py
```

После этого нужно ввести количество раз, сколько нажмётся `пробел`. Как только значение введено, через 3 секунды `пробел` будет нажиматься 30 раз в секунду...
